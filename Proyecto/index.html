Guia Rapida de Comandos Git

git init

Inicializa un nuevo repositorio GIt local en el directorio actual.

git add .

Agrega todos los archivos el area de preparacion (staging).

Git push

Envia los commits locales al repositorio remoto.

git status

Muestra el estado actual del respositorio, archivos modificados y no rastreados.

git commit -m "mensaje"

Guarda los cambios agregados con un mensaje decriptivo.

git pull

Obtiene y fusiona los cambios del repositorio remoto.
